# Cosmosis Navigator Launchers

Double-click launchers live here so the sketch root can stay focused on the Processing tabs.

## Cosmosis

- `Cosmosis/Cosmosis Navigator Fresh Start.app`
- `Cosmosis/Cosmosis Navigator Fresh Start.command`

Use these to start the main instrument stack from a clean slate.

BasinNET launchers are omitted from this first public release. See
`../../../docs/roadmap/basinnet.md`.

The `.pde` files intentionally remain in the sketch root because Processing treats them as sketch tabs.
