# Recursive Dimensional Field Theory Principle

Recursive Dimensional Field Theory, as used in this suite, is an experimental generative framework rather than a finished physical theory. The working principle is that complex observable structure can be explored through recursive field states, phase thresholds, holonomy, Floquet-style forcing, topology, and environmental perturbation.

In practice, RDFT is implemented as an instrument:

- A shared field state is evolved or sampled.
- Control parameters reshape depth, alpha, coherence, source pressure, Floquet drive, CTC bias, and topology.
- External data can act as a shaper rather than a deterministic truth source.
- Visual, sonic, and geometric surfaces expose different projections of the same underlying field vocabulary.

The suite is therefore best understood as an experimental language for dynamic systems: it lets users hear, view, wrap, print, and compare the behavior of recursive fields under different constraints.

## Important Framing

The code contains speculative physics language. It should not be read as peer-reviewed physical proof. The value of the project is in the instrument design, the procedural mappings, the exploratory topology, the sonification, and the generative research method.
