# Code of Conduct

This project is a research and creative-code space. Treat collaborators with patience, curiosity, and respect.

Expected behavior:

- Be constructive and specific.
- Credit prior work and collaborators.
- Separate critique of an idea from critique of a person.
- Make room for experimental language while keeping claims honest.
- Do not harass, threaten, demean, or target others.

Reports or concerns can be handled through GitHub issues or direct maintainer contact.
